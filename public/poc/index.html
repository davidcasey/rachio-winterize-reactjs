Rachio Winterize - Proof of concept

At the moment, this is all static—no data binding nor dynamic UI. Open your browser console to see logging.

  1. Enter API key and click the "Fetch" button
  2. Click "Winterize" to begin the blow out process
  3. Click "Cancel" anytime to stop the blow out process

Your Rachio API token is never stored on the server and only used for the duration of your session. Accessing the site over a public network is not recommended.
